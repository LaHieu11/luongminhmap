# Hướng dẫn thêm ảnh địa điểm

## Cách thêm ảnh

1. Đặt ảnh của bạn vào thư mục này (`frontend/public/images/`)
2. Đặt tên file theo format: `location-[id].jpg` hoặc `location-[id].png`

## Danh sách ID địa điểm

- `location-1.jpg` - Di tích lịch sử Đồn Đen
- `location-2.jpg` - Quảng trường 28 tháng 01
- `location-3.jpg` - Tượng đài anh hùng Hà Quang Vóc
- `location-4.jpg` - Di tích khảo cổ, lịch sử Rừng cò
- `location-5.jpg` - Ngân hàng Vietcombank Lương Minh
- `location-6.jpg` - Sân vận động Lương Minh
- `location-7.jpg` - Nhà máy chế biến thủy sản
- `location-8.jpg` - UBND xã Lương Minh
- `location-9.jpg` - Chợ Lương Minh
- `location-10.jpg` - Trạm Y tế xã Lương Minh
- `location-11.jpg` - Trường THCS Lương Minh

## Định dạng ảnh được hỗ trợ

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Kích thước khuyến nghị

- Tỷ lệ: 16:9 hoặc 4:3
- Độ phân giải: Tối thiểu 800x600px
- Dung lượng: Nên dưới 500KB để tải nhanh

## Lưu ý

- Nếu không có ảnh cho một địa điểm, hệ thống sẽ tự động sử dụng ảnh mặc định
- Ảnh sẽ được tự động copy vào build khi deploy lên Vercel
- Đảm bảo ảnh có quyền sử dụng hợp pháp
